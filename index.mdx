The Cursor Cookbook is a collection of ready-to-run examples showing how to embed Cursor's coding agent into your own applications. Each example demonstrates a real-world pattern — from a minimal Node.js script that streams agent output, to a full Next.js app with a live React preview, to a kanban board for managing cloud agents.
